## 📖 Overview

**Orlence OS** is a category-defining **Business Intelligence Layer** designed for founders and modern businesses. 

Most companies today suffer from disconnected data. Sales live in Shopify, payments in Paystack or Stripe, marketing in Instagram, and customer conversations in WhatsApp. Orlence OS unifies all these fragmented signals into a single, intelligent "Business Brain" that doesn't just show you charts—it tells you exactly **what happened, why it happened, and what you should do next.**

Instead of selling "AI," Orlence sells **business outcomes**: more revenue, less stress, better decisions, and time saved.

## 🛑 The Problem

Modern businesses don't need more software; they need software that understands everything. 

Today's reality:
* **Fragmented Tools**: You are forced to connect the dots manually between Instagram, Shopify, Paystack, Excel, WhatsApp, and your bank.
* **Information Overload, Insight Starved**: Traditional dashboards show you numbers (e.g., "Sales are down 15%"), but they don't explain *why* or offer actionable solutions.
* **Time Wasted**: Founders spend hours cross-referencing platforms instead of growing the business.

## 💡 The Solution & Value Proposition

Orlence OS sits above your existing tools as an intelligence layer. 

By connecting your commerce, payment, and messaging pipelines, Orlence provides a conversational, AI-driven operating system that acts as your digital COO.

### The Four Pillars of Orlence:
1. **Understand**: Know exactly what's happening across your entire business ecosystem in real-time. Track revenue, cash flow, customer cohorts, and inventory without switching tabs.
2. **Explain**: Discover *why* things are happening. Through Root Cause Analysis, Orlence can tell you that revenue dropped specifically because Instagram traffic fell and a best-seller went out of stock.
3. **Predict**: See tomorrow before it happens. Anticipate revenue forecasts, inventory depletion timelines, and customer churn risk based on historical and active patterns.
4. **Act**: Turn insight into action. Orlence can automatically draft restock purchase orders, launch WhatsApp reminder campaigns, or generate invoices for your approval.

## 🎯 Target Audience

Orlence OS is built for ambitious founders, SME owners, and modern enterprise leaders who manage multi-source commerce operations. 
* **E-commerce & Retail**: Brands managing online storefronts (Shopify, WooCommerce) alongside retail point-of-sale systems and social commerce.

## 🛠️ Tech Stack & Architecture

* **Frontend**: React 19, TypeScript, Vite
* **Styling**: Tailwind CSS (v4), Inter Font, Framer Motion (animations), Lucide React (icons)
* **AI Intelligence**: Google Gemini API (`gemini-3.5-flash`) for real-time data synthesis and conversational AI processing.
* **Backend**: Express.js server (local backend API proxy).

## 🚀 Getting Started (Local Development)

This repository contains everything you need to run Orlence OS locally.

**Prerequisites:** Node.js (v18 or higher recommended)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment:**
   Create a `.env` or `.env.local` file in the root directory and add your Google Gemini API key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

3. **Run the Development Server:**
   ```bash
   npm run dev
   ```

4. **Access the Application:**
   Open `http://localhost:3000` (or the port specified in your console) to view the Orlence Landing Page and OS Sandbox.
